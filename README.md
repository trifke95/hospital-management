# hospital-management
Hospital management system built as a Java console application. Features role-based access for Administrator, Doctor, and Patient. Built with layered architecture and manual dependency injection.

# Hospital Management System

Console-based hospital management application built in Java.

## Technologies
- Java 17
- Maven

## Features
- Role-based access (Administrator, Doctor, Patient)
- Patient and doctor registration
- Appointment scheduling
- Medical records and diagnosis tracking
